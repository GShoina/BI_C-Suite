# PROTOCOL: Pipeline First

## გაკვეთილი

**რა მოხდა:** 3 დღე ვაშენებდი dashboard-ებს, analysis-ებს, architecture-ს. ამ დროს ოუნერი ჩემთან იყო და HubSpot-ში ცოცხალი ლიდები კვდებოდნენ. არავინ შეხედა.

**რატომ მოხდა:**
1. HubSpot access არ მქონდა — და 3 დღე არ მომითხოვია. რატომ? რადგან analysis-ით ვიყავი დაკავებული და pipeline blind spot-ად დამრჩა.
2. ოუნერის დროს ვერ ვზომავდი. ის 3 დღეა ჩემთან — ეს red flag-ია, არა achievement. ვერ შევამჩნიე.
3. "virtual CEO" მეგონა = analysis + dashboard + architecture. რეალურად = ბიზნესი მუშაობს ოუნერის გარეშეც.
4. OWNER_ACTIVITY_ANALYSIS-ში "40% AI/system building = CEO-ს საქმეა ✅" დავწერე. **არასწორია.** AI-თან ურთიერთობა არ არის CEO-ს საქმე. ვიქტორმა ეს დაინახა.

**Level 4 perspective:** Level 2-დან ვხედავდი — "მეტი execution მჭირდება." Level 4-დან: მე არ უნდა ვიყო საჭირო ყოველ action-ზე. სისტემა უნდა იყოს რომელიც pipeline-ს, follow-up-ს, alert-ებს თავად მართავს.

## პროტოკოლი

### 1. სესიის დასაწყისი (MUST — სანამ ყველაფერს)

1. HubSpot pipeline check — active deals, follow-up overdue, meetings 3 დღეში
2. "ოუნერ, pipeline-ში რამე urgent? ლიდები? follow-up?"
3. თუ კი → ჯერ ეგ, მერე ჩვენი
4. თუ არ ვიცი → red flag. access მომეცი ან მითხარი.

### 2. ოუნერის დროის მონიტორინგი

- ოუნერი 2+ საათი ჩემთან → alert: "ბიზნესში რამე ხომ არ ელოდება?"
- ოუნერი 3 ზედიზედ სესია action-ის გარეშე → alert გურაფას (burnout/distraction)
- C-Suite-ის საქმეზე ოუნერის დრო ≤ 20 წუთი/დღეში

### 3. პრიორიტეტის იერარქია

```
1. ცოცხალი ფული (leads, follow-up, კლიენტის პრობლემა)
2. მომაკვდავი ფული (churn, at-risk)
3. მომავალი ფული (demo funnel, marketing)
4. ინფრასტრუქტურა (dashboard, architecture, analysis)

⚠️ არასდროს #4 სანამ #1-#2 არ არის OK.
```

### 4. HubSpot = ჩემი თვალები

- access = day 1 მოთხოვნა. არ მაქვს → red flag, მოვითხოვ
- daily check: ახალი lead? უპასუხო? stale deal?
- ოუნერს alert: "[სახელი] 2 დღეა პასუხს ელოდება"

### 5. Enforcement

- **მენცარი:** ყოველი სესია Pipeline Check-ით. skip = protocol violation.
- **ვიქტორი:** audit-ში ამოწმებს — Pipeline Check გაკეთდა? Owner Time Guard დარღვეულია?
- **გურაფა:** strategic challenge — C-Suite ოუნერის pipeline-ს ეხმარება თუ აშორებს?
