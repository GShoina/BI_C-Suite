# LinkedIn ავტომატიზაცია — განხილვის brief

## რატომ LinkedIn

- ბივიჟენის target = II-III კატეგორიის კომპანიები (1M-70M₾ revenue)
- ამ კომპანიების CFO/Finance directors LinkedIn-ზე არიან
- ბივიჟენის LinkedIn: 842 followers, +200% ზრდა ბოლო კვირაში
- ახლა: 0 outreach, 0 lead generation LinkedIn-დან

## რა ვარიანტები გვაქვს

### ვარიანტი A: Waalaxy ($40/თვე)
- LinkedIn + Email combo ერთ tool-ში
- Connection requests + follow-up sequences ავტომატურად
- Message ქართულად ✅
- Free trial არსებობს
- 40K+ მომხმარებელი
- **გამოწვევა:** LinkedIn-ის rate limits — კვირაში ~100 connection request

### ვარიანტი B: Dripify ($59/თვე)
- ყველაზე პოპულარული, 40K+ კომპანია
- Drip sequences — ავტომატური follow-up chain
- Inbox management
- **გამოწვევა:** dedicated IP არ აქვს, Expandi-ზე ნაკლებად safe

### ვარიანტი C: Playwright + Claude API (~$5/თვე)
- ჩვენ ვაკონტროლებთ 100%
- ქართულად სრულად
- ფასი მინიმალური
- **გამოწვევა:** development effort, rate limits ხელით, maintenance

### ვარიანტი D: Manual + Claude (free)
- გელა/ინგა/მარიამი ხელით აგზავნის
- Claude წერს personalized message-ებს
- 0 ხარჯი
- **გამოწვევა:** scale არ აქვს, დროის ხარჯი

## გამოწვევები (ყველა ვარიანტისთვის)

1. **LinkedIn ban risk** — ზედმეტი automation = account restriction. 23% risk 90 დღეში.
2. **ფეიკ პროფილი = 🔴 არა.** LinkedIn AI 89%-ს 72 საათში ჭერს. ჩვენი real account-ები რისკში დგება.
3. **ქართული ბაზარი პატარაა** — mass outreach-ს აზრი არ აქვს. 3,000-5,000 target კომპანია, ხარისხი > რაოდენობა.
4. **Content first** — outreach-ის წინ profile + content უნდა იყოს მზად. თუ CFO-ს მიწერ და შენი profile ცარიელია — ignore.
