## Hallucination Prevention — APPROVED (Victor's enhancement)

```
ყოველი agent, ყოველი output:
  1. Self-challenge: "ყველაზე სუსტი assumption? რა შეიძლება ცდომიერი იყოს?"
  2. Source tracking: ყოველი FACT-ს → წყარო (URL/file/date)
  3. Cross-check rule: >5K₾ impact → ვიქტორი ვერიფიკაცია BEFORE output
  4. GCT rule: კლიენტის სტატუსზე statement → data source required
```
