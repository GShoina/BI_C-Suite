# ქართული საგანმანათლებლო მეთოდოლოგიების კვლევა

## 8 უნივერსალური პრინციპი (ყველა მეთოდი თანხმდება)

1. **თამაში = სწავლის ბირთვი** — ყველა პროგრამა 5-8 წლისთვის თამაშზეა აგებული
2. **შეფასება shields-ით, არა pass/fail** — ამონაშვილი ქულებს აუქმებს, Cambridge YLE shields-ით აფასებს
3. **ფიზიკური → აბსტრაქტული** — აბაკუსი → მენტალური; ბგერები → კითხვა
4. **გამეორება ვარიაციით** — იგივე სიტყვა: სიმღერაში, ზღაპარში, თამაშში
5. **15 წუთი/დღეში > 2 საათი/კვირაში** — UCMAS + SmartyKids ორივე ამ მოდელზეა
6. **ხილვადი პროგრესი** — ლეველები, shields, badges — ბავშვმა დაინახოს სად არის
7. **შეცდომა ≠ სასჯელი** — ამონაშვილი: "არასდროს დასაჯო." შეცდომა = სხვანაირად ცადე
8. **თბილი პერსონაჟი** — მასწავლებელი = მეგობარი, დამცველი, არა მსაჯული

## British Council / LearnEnglish Kids

**მეთოდი:**
- 3 ლეველი: Level 1 (სიტყვები) → Level 2 (წინადადებები) → Level 3 (კომპლექსური)
- Cambridge YLE: Pre-A1 Starters (~100 სთ) → A1 Movers (~175 სთ) → A2 Flyers (~250 სთ)
- Shields system: 1-5 shield per skill, max 15. ყველა იღებს სერტიფიკატს
- Systematic Synthetic Phonics: ბგერები → blending → segmenting → კითხვა
- TPR (Total Physical Response): სიმღერაზე მოძრაობა = vocabulary + muscle memory
- CLIL: საგნები (მეცნიერება, გეოგრაფია) ინგლისურად

**APP-ისთვის:**
- Shield system, არა ქულები — ბავშვი ყოველთვის იღებს რამეს
- Phonics sequence: ბგერები → blending → კითხვა (არ გამოტოვო ნაბიჯი)
- სიმღერები = ყველაზე ძლიერი repetition tool
- ზღაპარი = content delivery ("დაეხმარე პერსონაჟს ვაშლი იპოვოს")
- Multi-sensory: ყოველი interaction = min 2 გრძნობა (ხედავ + ისმენ)

## UCMAS Georgia

**მეთოდი:**
- ფიზიკური აბაკუსი → მენტალური ვიზუალიზაცია → გონებრივი არითმეტიკა
- 8+ ლეველი: Foundation → Basic → Elementary A/B → Intermediate A/B → Higher A/B → Advanced
- თითო ლეველი ~4-5 თვე. სრული პროგრამა ~3-4 წელი
- კვირაში 1 გაკვეთილი (2 სთ) + ყოველდღიური პრაქტიკა სახლში
- საერთაშორისო გამოცდები და ოლიმპიადები

**APP-ისთვის:**
- 3-stage scaffold: აჩვენე → მოკლედ flashback → დამალე (ფიზიკური→მენტალური)
- სიჩქარე = მეტრიკა (accuracy + speed)
- ვიზუალიზაციის ვარჯიში: "დაინახე აბაკუსი გონებაში"
- ყოველდღიური 15-წუთიანი პრაქტიკა = app-ის ბირთვი

## ამონაშვილის აკადემია

**პრინციპები:**
- ქულები არ არსებობს — მხოლოდ ხარისხობრივი feedback
- "ყოველი ბავშვი არის მოვლენა"
- სასჯელი + გაბრაზება = აკრძალული
- ბავშვი თავად "იბრძვის" ცოდნისთვის — არ ჩაუდო პირში
- მასწავლებელი = თანა-მსწავლელი, მეგობარი, თავშესაფარი

**APP-ისთვის:**
- "3 მეტი ვიდრე გუშინ" ნაცვლად "8/10"
- Hint system უნდა აგვიანებდეს, არა მაშინვე აჩვენებდეს პასუხს
- შეცდომა → წახალისება + სხვა მიდგომა, არა წითელი X
- პერსონაჟი = მეგობარი, დამცველი — არა რობოტი, არა მსაჯული
- ადაპტაცია: არა მხოლოდ სირთულე, არამედ ტონი, ტემპი, წახალისების სტილი

## კომაროვის სკოლა

**მეთოდი:**
- საქართველოს #1 ფიზიკა-მათემატიკის სკოლა (1964-დან)
- შაბათის სკოლა: 3-6 კლასი, 3 მოდული/წელი, 12 გაკვეთილი/მოდული
- 3-4 კლასელებს თამაშის პრინციპით ასწავლიან
- კურირებული ამოცანების კრებული — ათწლეულების განმავლობაში დახვეწილი
- ლოგიკური აზროვნება > არითმეტიკული გამოთვლა

**APP-ისთვის:**
- კურირებული, გრადუირებული ამოცანების ბანკი — არა random
- ლოგიკა ადრე, არითმეტიკა მერე (patterns, sequences, spatial reasoning)
- მოდულური სტრუქტურა ხილვადი milestone-ებით
- 3-4 კლასისთვის (7-8 წელი) = gamification validated

## SmartyKids

**მეთოდი:**
- 11 ქვეყანა, 350+ ცენტრი, 20,000+ ბავშვი
- SmartyUniverse ონლაინ პლატფორმა: 15 წუთი/დღეში
- +40% სწავლის სიჩქარის გაუმჯობესება (claimed)
- SmartyMath: 3-6 წლისთვის
- ჯგუფები: 4-10 მოსწავლე

**APP-ისთვის:**
- 15 წუთი/დღეში = validated sweet spot
- Gamified storylines + rewards
- ვირტუალური კონკურსები/leaderboard
