## რა ვერ უნდა გავიმეოროთ (Gegidze weak spots)

- **Stock testimonials bulk** (name + generic praise) — Bivision-ს რეალური named testimonials სჭირდება
- **Unsourced metric claim** ("3X ROI!") — Bivision ვერ ხდის ამას: ციფრი + წყარო-წელი
- **English-primary** — Gegidze-ს foreign audience ICP. Bivision = Georgian market, Georgian primary
- **7-services umbrella** — Bivision BI specialist, focused. Don't drift

## Mentari-ს 10 სამოქმედო რეკომენდაცია

1. **ყოველი CTA ≤7 სიტყვა** + action verb + specific offer. "Contact us" forbidden forward.
2. **Time-commit visible ყოველ asset-ზე:** "N კვირაში · Y რესულტი" format. Hero, bio, email signature.
3. **DM trigger word** — low-friction entry. Primary: "dashboard", "audit", "3 კვირა". Post-ში explicit: "Message me '[trigger]' for [outcome]"
4. **Case study 1-liner format:** `[Client] — [Before → After] — [Year]`. ყოველი LinkedIn slide, landing page card, email snippet.
5. **Price anchor visible** — არ დამალო. Range OK, mystery არა. Proposal cover + site services + FB ad footnote.
6. **Logo wall 5 top client** — persistent (site header, LinkedIn banner, proposal cover, FB bio): TBC Pay, BOG, Aversi Pharma, NRG Holding, Orbita.
7. **ქართული primary, English supplementary** — Bivision ICP ქართული; Georgian hero + EN secondary only.
8. **Video / screen-demo visual** — არა stock robot handshake. 15-30 წამი actual Qlik dashboard tour. FB ad creative + landing page hero.
9. **Owner voice direct** ("Gela ვამბობ..."): LinkedIn long-form + founder newsletter. Gegidze-ს ამას ვერ ექნება (49-person scale).
10. **Bundle objection defense:** ყოველ proposal-ში section — "Why not buy BI as part of automation bundle?" → because BI depth უწევს specialist-ს, bundle generalist-ს.

## Anti-patterns — გაფრთხილების ლისტი

| Anti-pattern | რატომ forbidden |
|--------------|----------------|
| "Contact us" generic CTA | 0 action triggering |
| "3X ROI!" source-less | credibility leak |
| Stock AI robot images | contradicts Bivision depth story |
| Buzzword soup (synergy, cutting-edge, innovative) | content-void |
| English-primary Georgian market-ზე | wrong audience match |
| Bulk testimonials name-only | trust signal weak |
| Vague pricing | friction without benefit |

## Messaging 4-slot framework (ყოველი post / ad / email)

```
1. HOOK     — pain წერტილი ერთ წინადადებად  (7-15 სიტყვა)
2. COMMIT   — specific time + outcome       ("3 კვირა-ში dashboard")
3. PROOF    — named-client-number-year      ("TBC Pay 3 დღე → 15 წთ, 2024")
4. MICRO-CTA — ≤7 სიტყვა, DM trigger word   ("DM 'dashboard'")
```
