# Sales Learning Skill — Self-Improving

## Purpose
ყოველი sales interaction-ის შემდეგ ეს skill განახლდება რეალური გამოცდილებით.
აგენტი (Mentari) ყოველ session-ზე კითხულობს დაგროვილ lessons-ს და სწავლობს.

## როგორ მუშაობს
1. Owner/Mentari ატარებს prospect interaction-ს (შეხვედრა, call, email)
2. შედეგი იწერება §Experience Log-ში (რა მოხდა, რა იმუშავა, რა არა)
3. ყოველი 5 entry-ის შემდეგ → §Patterns განახლდება (რა ტენდენცია ჩანს)
4. მომდევნო prospect-ზე აგენტი Patterns-ს კითხულობს და ადაპტირდება

## Current Playbook

### Objection Handling
| Objection | Default Response | Learned Adjustment |
|---|---|---|
| "ძვირია" | ROI framing: "850₾ = 2 სამუშაო დღე" | (ჯერ არ გვაქვს data — პირველი 3 prospect-ის შემდეგ განახლდება) |
| "3 თვის მერე" | Pilot proposal: "დავიწყოთ ახლა, 3 თვეში შედეგი" | (pending) |
| "გუნდს ვკითხავ" | Brief co-creation: "ერთად მოვამზადოთ" | (pending) |
| "სხვისი მოლოდინი" | Expectation alignment: "კონკრეტულად რა ელოდე?" | (pending) |

### Proposal Effectiveness
| Element | Hypothesis | Result | Learned |
|---|---|---|---|
| Problem-first opening | უკეთესია ვიდრე "მადლობა" opening | ❓ UNTESTED | v2 template მზადაა, მომდევნო prospect-ზე ტესტი |
| ROI framing in price section | "2 დღის ღირებულებაზე ნაკლები" | ❓ UNTESTED | (pending) |
| 3-month Free Cancellation emphasis | risk objection-ს ხურავს | ❓ UNTESTED | (pending) |
